3.WEB Hosting


1) 깃허브(GitHub)사이트 접속 [111p]
2) New repository > Repository name 작성 > Public 체크 > Initialize this repository with a README 체크 > Create repository [112p]
3-1) Code > Upload files > choose your files > 업로드할 파일 모두 선택 후 열기 > 보관내용 작성(작은 빈칸) > Commit changes [113p]
3-2) 파일을 수정하고 싶을때 : 업로드 > 수정한 내용 적기 > Commit changes [113p]
4) Settings > GitHub Pages의 None을 master branch로 바꾸기 > 웹 사이트 주소 생성 [116p]
5) BitNami WAMP Stack 사이트 접속 > Win / Mac / Linux > Downlord for Windows 64-bit > No thanks, just take me to the downlord [124p]
6) 설치 파일이 실행되면 Next > PhpMyAdmin만 체크 후 Next > Next > 비밀번호 설정 후 Next > Launch Wampstack in the cloud with BitNami 체크 해제 후 Next > 방화벽이 나타나면 Allow access > Launch BitNami WAMP Stack 체크 후 Finish [127p]
7-1) 윈도우탐색기 > BitNami > Wampstack-7.1.20-0 > manager-windows > Manage Servers > Apache Web Server가 초록색이면 웹 서버가 켜져있음을 의미 [132p]
7-2) pache Web Server가 빨간색이면 웹 서버가 꺼져있음을 의미 > Start와 Stop으로 켰다 껐다 할 수 있음 [132p]
8) 인터넷 주소창에 http://127.0.0.1/(내가 만든 웹 페이지 이름)으로 들어가기 [135p]
9) 윈도우탐색기 > BitNami > Wampstack > apache2 > htdocs(웹 페이지가 저장된 폴더라는 것을 의미) > (내가 만든 웹 페이지 이름)이 존재 [135p]
10) htdocs에 들어있는 모든 파일 삭제 > 내가 만든 모든 웹 페이지를 복사해서 htdocs에 붙혀넣기 [138p]
